# bengaluru-traffic-analytics
Power BI project analyzing Bengaluru traffic patterns, congestion, and trends using data visualization.

## Bengaluru Traffic Analytics (Power BI)

## Project Overview
This project analyzes traffic patterns in Bengaluru using Power BI. It provides insights into congestion levels, traffic volume, and travel time across different areas. The dashboard helps understand how factors like weather and road construction affect traffic.

## Objectives
- Analyze traffic congestion across major areas.
- Study impact of weather on traffic.
- Evaluate road performance.
- Understand public transport usage vs congestion.
- Identify traffic trends over time.

## Dataset Features

- Area Name (Electronic City, Hebbal, etc.)
- Traffic Volume
- Travel Time
- Congestion Levels
- Weather Conditions
- Road Data

Key Insights

High congestion in Koramangala and M.G. Road
Weather affects traffic volume
Road construction increases congestion
Public transport usage impacts traffic

## Tools Used
- Power BI
- Data Analysis
- Data Visualization

## Dashboard Highlights
- Traffic Volume by Area
- Congestion Analysis by Road/Intersection
- Impact of Weather Conditions
- Public Transport vs Congestion Analysis
- Monthly Traffic Trends
